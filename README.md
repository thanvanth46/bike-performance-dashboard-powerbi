# bike-performance-dashboard-powerbi
Power BI dashboard analyzing bike performance metrics such as price, mileage, power, torque and user ratings using Excel dataset.

Project Description
This project focuses on analyzing and visualizing bike performance data using Microsoft Excel and Power BI. The main objective of the project is to transform raw motorcycle data into meaningful insights through data cleaning, data modeling, and interactive dashboards.

The dataset contains information about different bike brands and models including attributes such as engine capacity (CC), power (BHP), torque (Nm), mileage, weight, top speed, fuel tank capacity, service cost, and user ratings. Using Excel, the data was organized and prepared for analysis. After preprocessing the data, it was imported into Power BI to build an interactive dashboard.

The Power BI dashboard provides clear visual insights into various bike performance factors. It helps users compare motorcycles based on specifications, performance, and cost efficiency. Different visualizations such as bar charts, comparison charts, and performance indicators are used to make the data easy to understand.

This project demonstrates practical skills in data analysis, data visualization, and dashboard development, which are essential skills for roles such as Data Analyst or Business Analyst. It also shows the ability to convert raw data into actionable insights using modern data tools.

Tools Used:

Microsoft Excel (Data Cleaning & Preparation)

Microsoft Power BI (Data Visualization & Dashboard)

Key Features:

Data cleaning and structuring using Excel

Interactive Power BI dashboard

Bike performance comparison across multiple brands

Visual analysis of mileage, power, torque, and pricing factors
